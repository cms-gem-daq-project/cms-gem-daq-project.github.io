    Welcome to the CMS GEM DAQ Project GitHub Page.

    Code hosted here includes the firmware for our main boards (the GLIB and the OptoHybrid) as well as the online software to run the system and integrate with the CMS online system To begin development please first fork the repository of your choice (e.g., gem-daq-code) and then set up your development area:

    $ cd <some directory where you will base your development>
$ git clone git@github.com:cms-gem-daq-project/gem-daq-code.git

    If you have already checked out the code

    $ cd <some directory where you will base your development>/gem-daq-code
$ git fetch origin
